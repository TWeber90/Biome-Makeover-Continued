# Biome Makeover Continued

A community-maintained continuation of the archived Fabric version of **Biome Makeover** for Minecraft **1.20.1**.

The goal of this project is to preserve the original gameplay while providing important bug fixes and compatibility improvements.

## Changes

### 1.20.1-1.11.4-kod.1

- Fixed a crash when using Accessories with projectile resistance.
- No gameplay changes.
- Drop-in replacement for the original Fabric release.

## Installation

Replace the original **Biome Makeover** mod with this version.

Do **not** install both at the same time.

## Compatibility

Minecraft:
- 1.20.1

Loader:
- Fabric
